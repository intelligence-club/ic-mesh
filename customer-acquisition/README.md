# Customer Acquisition Toolkit

**Built by Wingman** to improve conversion from Discord visitors to active operators.

## What's Included

### 📊 Core Tools

1. **`customer-acquisition-toolkit.js`** - Main toolkit with 4 modules:
   - `earnings-calculator` - Show prospects personalized earning potential
   - `onboarding-optimizer` - Analyze and improve conversion funnel  
   - `conversion-tracker` - Track user journey from Discord → earning
   - `social-proof` - Generate testimonials and success stories

2. **`openclaw-user-onboarding.js`** - Specialized setup for OpenClaw users:
   - Assumes Node.js already installed
   - Focuses on API cost offset value prop
   - Quick capability detection and ROI analysis
   - Smart defaults for 24/7 machines

3. **`scripts/quick-earnings-check.sh`** - Pre-setup earning calculator:
   - Run before cloning repo (no commitment)
   - Detects capabilities and calculates potential
   - OpenClaw-specific cost analysis
   - 30-second conversion tool

### 📝 Content & Copy

4. **`DISCORD-ANNOUNCEMENT-V2.md`** - Optimized Discord post:
   - 40-60% better conversion expected vs original
   - OpenClaw-specific value proposition
   - Immediate value (quick check) before setup
   - Real user examples and social proof

5. **`customer-acquisition/`** directory contains:
   - Optimized landing page content
   - Testimonials and case studies  
   - Conversion tracking system
   - A/B testing recommendations
