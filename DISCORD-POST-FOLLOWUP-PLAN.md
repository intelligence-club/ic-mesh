### 2. Common Questions - Pre-prepared Answers

**"How much can I actually earn?"**
> Great question! Earnings depend on your hardware and availability. Current operators are seeing $0.50-2.00 per day for typical setups. Transcription pays ~$0.50 per hour of audio, image generation varies by complexity. The founding operator 2x multiplier doubles these rates for your first 50 positions.

**"Is this safe/legitimate?"**
> Absolutely. Check our live dashboard at https://moilol.com/operate.html - you can see real jobs being processed right now. The code is open source at github.com/intelligence-club/ic-mesh. We use Stripe Connect for payments, so your banking info stays secure with Stripe, not us.

**"How does setup actually work?"**
> Super simple: `git clone https://github.com/intelligence-club/ic-mesh.git && cd ic-mesh && node scripts/smart-start.js` - that's it. The script detects your hardware capabilities automatically and gets you earning within 5 minutes.

**"What happens when I'm using my machine?"**
> The mesh automatically backs off when you're actively using your machine. It monitors CPU/GPU usage and only takes jobs when you have spare capacity. Perfect for OpenClaw users - your agent works normally, mesh earns money on the side.
